# GoalFlow — Personal Goal Management

GoalFlow is a premium, opinionated goal-tracking Kanban platform built with React, Vite, Tailwind CSS, DnD Kit, and Zustand.

## 🚀 Quick Start

1. Install dependencies:
   ```bash
   npm install
   ```
2. Start the development server:
   ```bash
   npm run dev
   ```
3. Open the provided local URL (typically `http://localhost:5173`).

---
*Note: This project relies on local state persistence (`localStorage`). If you wish to reset the application to its original seed data, navigate to Settings > Advanced > Factory Reset inside the application.*
